# 🏥 Hospital Management System (Java)

A console-based Hospital Management System developed using Java. This project demonstrates Object-Oriented Programming (OOP) concepts and provides a simple way to manage patients, doctors, and appointments.

---

```text
      HOSPITAL MANAGEMENT SYSTEM
========================================

2. View Patients
3. Add Doctor
4. View Doctors

6. View Appointments
7. Search Patient
8. Delete Patient
9. Exit

Enter your choice:
```

## 🚀 Getting Started

### Prerequisites

- Java JDK 8 or above
- VS Code / IntelliJ IDEA / Eclipse

### Clone the Repository

```bash
git clone https://github.com/your-username/hospital-management-system-java.git
```

### Navigate to the Project Folder

```bash
cd hospital-management-system-java
```

### Compile the Project

```bash
javac HospitalManagementSystem.java
```

### Run the Application

```bash
java HospitalManagementSystem
```
